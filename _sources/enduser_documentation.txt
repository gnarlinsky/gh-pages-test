**Viewing all customers**
-------------------------
After logging in at `http://localhost:8000/admin/` (username and password
"homer"), 

.. image:: images/login.png
   :width: 220 px


you will be taken to the main admin screen, where you can view the
current signed-up customers. Here you can sort the columns and search on any
field.

.. image:: images/admin_change_list_whole_screenshot.png
   :width: 600 px

**Editing customers**
----------------------
Clicking on a name on the main admin screen (see above) takes you to that
customer's individual page. Here you can edit the customer's information.

.. image:: images/admin_change_form_screenshot.png
   :width: 600 px

**Deleting customers**
----------------------
You can delete customers' records from either the individual customers' pages,
on at a time or from the main admin screen. Here you can also *batch delete*
