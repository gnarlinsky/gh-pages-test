Unit tests
==================

Each ``*.py`` in the ``tests`` directory, as well as various helper files, have
associated ``test_*.py`` files in the ``tests`` directory, e.g. the unit tests
for ``views.py`` are in ``tests/test_views.py``.

Lines like ``t_info('Is empty string parsed correctly?', 5)`` provide
additional documentation. See `ttools.py
<demo-other.html#module-app.tests.ttools>`_ for more information.
